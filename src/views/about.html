wools++

About

wools++ is a community-driven and fan-made project. The part that does stuff is written in python, and it uses jinja2 with bootstrap html/css/javascript templates for displaying. wools++ uses BeautifulSoup to collect data from users' profiles at Project Ares and matplotlib to produce the graphs. The project is hosted on Google's App Engine. All of the source code can be found here.

The project is currently in its alpha stages, so it will still be some time before I get things straightened out. This means I may have to clear the data every once in a while. Sorry about that!

FAQ

My stats aren't updating!
wools++ only collects data a couple times per day. This is to limit requests made to Project Ares' website, as well as to limit the work that the app has to do. The frequency at which data is collected may change in the future. Remember, wools++ is more about "long term" stats than up-to-the-second ones.
My stats aren't accurate / I found an arbitrary problem!
Please report all problems in the official wools++ thread on Project Ares' Forum. (No thread yet)
The graphs aren't really that interesting.
Give it time! You won't see much at first. Play for a few weeks and it should get better.
Can I put one of these graphs on my Project Ares profile?
Yes! That's mostly the point of this app - to generate graphs for your own profile on Project Ares!
What's a rolling stat?
A rolling stat is a stat that is computed with only a given amount of recent data. For example, a 2 day rolling KD would be computed as (kills earned in the last 2 days) / (deaths earned in the last 2 days). As of now, wools++ computes rolling stats based on the last 7 days of playing time (if you haven't been in the system for 7 days, it's calculated from the day you first joined). This time period is totally arbitrary, but it seems to be okay. One of the advantages of a rolling stat is that it gives a player feedback much more quickly than a standard statistic that includes everything, such as (all kills)/(all deaths). More info here.
What's a "RWD"?
Common abbreviations used:
  • K=Kills
  • D=Deaths
  • C=Cores
  • W=Wools
  • M=Monuments
  • O=Objectives
  • S=Scoreboxes (just kidding!)
  • R=Rolling
Most stats are computed as something/deaths. So, KD would be kills/deaths. "RWD" means "rolling wools deaths" or (wools gotten in the past 14 days)/(deaths in the past 14 days)
