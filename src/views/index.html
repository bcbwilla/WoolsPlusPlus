Welcome to wools++!

wools++ provides players of the Overcast Network (OCN) with additional gameplay statistics including rolling values, objectives/deaths, time series plots, pie charts and server-wide comparative metastats.

This app is currently being revamped. Everything from the old version is still available for viewing, but data is not currently being collected. If you have a moment, please fill out this survey to help guide the development process. Thanks! Check back soon!

Start collecting data today!

Add yourself to the database to start collecting data! A couple times each day, wools++ pulls data from your profile at OCN and adds it to the database here. It then calculates some statistics and makes some plots.
